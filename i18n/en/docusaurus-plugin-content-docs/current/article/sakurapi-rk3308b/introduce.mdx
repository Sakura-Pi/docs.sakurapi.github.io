# Introduce

Sakura Pi RK3308B is our first officially released SBC.  
It is a low-power, cost-effective development board powered by the Rockchip RK3308B SoC.

![board-shot](./img/board-shot.webp)

### Processor

Thanks to the ARM® Cortex-A35 processor, the RK3308B boasts numerous advantages such as low power consumption, low cost, and minimal heat generation.

In modern mobile device architectures, the Cortex-A35 often serves as a 'little core' (efficiency core/E-core), striking a graceful balance between handling complex tasks and maintaining long standby times with its efficient performance.

![rk3308b-diagram](./img/rk3308b-diagram.jpg)

### Features
The basic specifications of the Sakura Pi RK3308B are as follows
- Processor: RK3308B
- Memory: 512MB DDR3
- Wireless WLAN: Supports IEEE 802.11a/b/g/n/ac, dual-band 2.4GHz/5GHz
- Bluetooth: Supports BLE 4.2
- eMMC: Optional 0GB / 8GB / 16GB
- Indicator LEDs: WS2812 (SPI) × 4
- LCD Interface: RGB666

### Pinout

The Sakura Pi RK3308B features a 40-pin header compatible with the Raspberry Pi Zero W.
**By default, U-Boot and the kernel use UART2 as the output interface.**

![rk3308b-pinout](./img/board-pinout.svg)

#### LLVMPipe GLSL
:::info
GitHub Repository **[astefanutti/kms-glsl](https://github.com/astefanutti/kms-glsl)**  

kms-glsl can rendering glsl shaders via GPU，but RK3308B doesn't has a GPU hardware
thus causes Linux forces use LLVMPipe to render it.
:::
We use LLVMPipe to render glsl shaders, conducting stress tests at a CPU frequency of 1.3GHz. Without any additional cooling, the junction temperature reaches around 140 degrees Celsius.
